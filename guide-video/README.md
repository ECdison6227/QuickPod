# QuickPod Guide Video

Remotion onboarding video project for QuickPod.

## Included

- `QuickPodGuide`: 1280x720, 30fps, 780 frames
- 5 scenes:
  - Welcome / overview
  - Notification permission setup
  - Break reminder activation
  - Anti-sleep workflow
  - Quick switcher hotkey demo

## Commands

```bash
npm install
npm run dev
npm run lint
npm run render:still
npm run render:guide
```

## Outputs

- Still preview: `out/guide-still.png`
- MP4 render: `out/quickpod-guide.mp4`
